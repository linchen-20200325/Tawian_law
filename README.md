# 🏛️ 台灣遺產稅・特留分・保險傳承規劃系統

依 **2026（115 年度）** 台灣《遺產及贈與稅法》、《民法》繼承編、《保險法》§112 之**確定性演算法**，
自動化計算遺產稅、特留分，並產出保險節稅與補償規劃建議。

> ⚠️ 本工具僅供試算參考，不構成稅務或法律意見。實際申報請洽會計師、地政士或律師。

## ✨ 功能（四階段演算法）

1. **淨資產與遺產總額計算** — 現金／股票／土地公告現值／房屋評定價格 − 負債 − 喪葬費 138 萬（負值歸零＝限定繼承）。
2. **2026 遺產稅精算** — 免稅額 1,333 萬、配偶扣除 553 萬、子女扣除 56 萬/人；10 / 15 / 20% 累進級距（累進差額 281.05 萬・843.15 萬）。
3. **民法應繼分・特留分診斷（完整四順位）** — 依民法 §1138／§1144 判定實際繼承順位：① 子女 ② 父母 ③ 兄弟姊妹 ④ 祖父母，配偶恆並存（與①平均分、與②③配偶 ½、與④配偶 ⅔）；特留分依 §1223（卑親屬/父母/配偶 ½、兄弟姊妹/祖父母 ⅓）。含重組家庭前婚子女診斷與特定人特留分總額。
4. **自動化保險傳承方案** — 策略Ａ 預留稅源保單（保額 = 遺產稅）、策略Ｂ 特留分補償保單（保額 = 特留分總額，依保險法 §112 不列入遺產）。

另含「實質課稅原則警示紅線」：資產 ≥ 1 億或年齡 ≥ 75 歲時，提醒改採分期繳保單並結合每年 244 萬分年贈與。

## 🚀 本機執行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ 部署（Streamlit Community Cloud）

1. 將本 repo 推上 GitHub。
2. 於 [share.streamlit.io](https://share.streamlit.io) 新增 App，指定 `app.py`、分支 `main`。
3. 不需任何 API 金鑰或 Secrets（純離線確定性計算）。

## 🧱 架構（SSOT）

所有稅法／民法數字集中於 `app.py` 頂端「法定常數區塊」，為單一真實來源；
資料流為 **輸入 → 純函式（`calc_net_estate` / `calc_estate_tax` / `calc_inheritance` / `calc_insurance_plan`）→ UI**，
無持久化、無外部 I/O，時間與空間複雜度皆為 O(1)。
